The code for Lab 1 is in client.java. To test:

1. 	Navigate to the folder containing echo.php and run 
	from command prompt using: php -S 0.0.0.0:8000 -t .

2. 	Open a new command prompt window and navigate to the	
	folder containing client.java and compile using:
	javac client.java

3. 	Once compiled run the command: java Client String1 String2
	..where String1 and String2 are the strings you wish to 
	send to the server and subsequently receive back in
	caps.
